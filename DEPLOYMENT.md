# Инструкция по деплою (развертыванию) сайта

Этот проект является Full-stack приложением (React + Express). Для его работы требуется хостинг с поддержкой Node.js.

## 1. Настройка Telegram бота

1. **Создание бота:**
   - Найдите `@BotFather` в Telegram.
   - Выполните `/newbot`, следуйте инструкциям.
   - Сохраните **API TOKEN**.

2. **Получение Chat ID:**
   - Найдите `@userinfobot` в Telegram.
   - Нажмите "Start", он пришлет ваш **Id**.

## 2. Настройка переменных окружения

На вашем хостинге в разделе "Environment Variables" (или в файле `.env` на сервере) добавьте:

```env
TELEGRAM_BOT_TOKEN=ваш_токен
TELEGRAM_CHAT_ID=ваш_id
NODE_ENV=production
```

## 3. Команды для запуска

При развертывании хостинг должен выполнить следующие шаги:

1. **Установка библиотек:**
   ```bash
   npm install
   ```

2. **Сборка фронтенда:**
   ```bash
   npm run build
   ```

3. **Запуск сервера:**
   ```bash
   npm start
   ```

## 4. Почему это важно?
Сайт использует сервер на Node.js для:
- Безопасной отправки данных в Telegram (чтобы никто не украл ваш токен бота).
- Работы API маршрута `/api/send-lead`.
- Раздачи статических файлов (HTML/JS/Images) из папки `dist`.

Если вы просто загрузите файлы на обычный PHP-хостинг через FTP, Telegram работать не будет. Используйте VDS или современные облачные платформы (Amvera, Render, Railway, DigitalOcean).
